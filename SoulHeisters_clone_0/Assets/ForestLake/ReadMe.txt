Thank you for the purchase! By default the asset made using build in pipeline

To change the SRP please follow the instruction

URP
Create URP Project, import pack
Unzip URP pack by double click it, press import
Import Water fix if it shown pink

For Unity 6.3 You can import FL_URP_Unity6.3 its includes more realistic water rendering that available only in Unity 6.3 

Check Readme for URP setup when imported

HDRP
Create HDRP Project, import pack
Unzip HDRP by double click it, press import
Apply Unity 6 fix if you on Unity 6+ version

Check Readme for HDRP setup when imported
